About Dunia

Dunia in Kiswahili means - The World Dunia is a memory game that allows the user to match cards based on memeory of where they are positioned on the game board. The cards faces are flags of various European countries. However this particlular project will be based on europe. This game board consists of 12 pairs cards, giving the user 24 cards in total to play with.

How to Play

To play, simply click on a card, to reveal its front face(image of a flag). If two cards are correclty matched, the user will score 1 point. When all cards are matched, the user was won the game and can start playing again.
